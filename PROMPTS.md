# Summary Notes

The results page includes a short written summary based on the audit output.

## Inputs Used

- number of tools
- team size
- use case
- total monthly savings
- total annual savings
- strongest recommendation

## Writing Goals

- keep it short
- include the main savings figure
- mention the strongest recommendation
- make it readable for a founder or finance lead

## Why It Is Separate

The summary logic lives outside the audit engine so the savings calculations stay focused on numbers and recommendation rules.
